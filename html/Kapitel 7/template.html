Was ist das Besondere an Tabellen in HTML?

Bei der Entwickelung von Html stand anfangs nur der Austausch Dokumenten im Vordergrund.. Dabei kümmerte sich niemand sonderlich um Layout oder Design - der Inhalt war wichtig, nicht die äußere Form. Deshalb ist es bis heute mit den Bordmitteln von HTML nicht möglich, Objekte(texboxen oder Bilder ) exakt auf einer Seite zu positionieren.

Als es im Zuge der Kommerizialisierung des Internets plötzich notwendig wurde, HTML-Seiten an das Corporate Design von Unternehmen anzupassen lag die Lösung in Tabellen. Ursprünglich schlicht für tabellarische Auflistungen konzpiert, kann eine in HTML erzeugte Tabelle praktisch jede Art von Elementen enthalten: Text, Bilder, Animationen, Flash Videos, und sogar weitere Tabellen.

Der erhebliche Nachteil von Tabellen liegt allerdings darin, dass ihre Bearbeitung äußerst zeitaufwändig is. Sollen die Seiten aktualisiert werden, müssen Sie u.U. die komplette Tabellenkonstruktion umbauen.

Diese Zeiten sind zum Glück vorbei. Heutzutage können Sie mit Css jedes Element beliebig auf Ihreer Seite platzieren.

Das Grundprinzip

Eine Tabelle in HTML beginnt stets mit dem Tag <table> und endet mit </table> Danach werden die einzelnen Zeilen de Tabelle notiert. Eine Zeile wird durch <tr> definiert und mit </tr> abgeschlossen.

Innerhalb jeder Zeile notieren Sie dann die einzelnen Zellen dieser Zeile mit <td></td>. Dabei muss jede Zeile zwingend dieselbe Anzahl an Zellen haben! Erst zwischen diesen td-Tags steht die eigentliche Information(Text, Bilder u,ä), die in dieser Zelle erscheinen soll.
