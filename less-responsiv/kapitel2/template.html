Kapitel 2 Variablen.

Variablen definieren

In CSS müssen Sie den Hex-Wert für die Farbe jedesmal, wenn Sie sie verwenden, von neuem notieren. Vor allem aber müssen Sie sie bei einer etwailigen Änderung an jeder Stelle in Ihrem CSS-Dokument ändern. Dies ist aufwendig, mühsam und vorallem fehleranfällig.

Mit LESS können Sie diese Farbe an zentraler Stelle in Ihrem Dokument einmalig als Variable anlegen und dieser dann beliebig vielen Eigenschaften als Wert zuweisen Wenn Sie eine Farbe ändern möchten, ändern Sie nur den Wert der Variablen. Der LESS-Compiler erzetzt dann überall im resultierenden CSS-Dokument dynamisch die Varibale durch den entsprechenden Farbwert.

Da sich LESS so eng wie möglich an die "echte" CSS-Syntax hält, werden Variablen gebildet @-Regeln.

@fontColor: #333

Mehrere Werte für eine Variable

Genauso wie CSS-Eigenschaften können LESS-Variablen auch mehrere Werte enthalten

BSP: @basefront: "Lucida Grande", "Lucida Sans Unicode", "Lucida Sans", Verdana, Helvetica, Arial, sans-serif

Auch folgender Code wäre möglich.

BSP: @myShadow: 2px 2px 4px rgba(0,0,0,0.4)
