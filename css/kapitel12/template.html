Kapitel 12 Positionierung von Elementen.

Trivia

Mit CSS können Sie jedes Element exakt auf Ihrer Seite positionieren. Dabei haben sie mehrere Möglichkeiten. Die beiden am häufigsten benutzten sind die statische und die absolute Positionierung. Daneben gibt es noch die relative und die fixierte Positionierung

Zunächst müssen Sie im Attribut position angeben, welche dieser Positierungsmethoden sie verwenden. Beachten Sie, dass Sie damit noch nicht bestimmen, wo genau sich ihr Objekt au der Seite befindet. Sie legen damit lediglich fest, welche Methode Sie zur Positionierng verwenden, d.h auf welche Weise das Objekt positionert werden soll.

Beispiel

img#logo{position: static;}

img#logo{position: relative;}

img#logo{position: absolute;}

img#logo{position: fixed;}

Statische Postionierung

Die statische Positionierungsmethode ist die Standard Methode, die verwendet wird, wenn Sie nichts anderes angeben. Das bedeutet, diese Methode haben wir bereits die ganze Zeit über verwendet. Sie müssen die Eigenschaft position:static i.d.R nicht explizit angeben.

Mit dieser Methode werden Objekte positioniert, indem sie sich gegenseitig verdrängen und damit an die gewünschte Stelle "drücken", ähnlich wie etwa Blätter auf einer Wasseroberfläche. Sie verwenden dafür üblicherweise Eigenschaften wie width, margin oder float

Relative Positionierung

Ähnlich funktioniert die relative Positionierung. Allerdings können Sie mit dieser Methode ein Element mit einem oder mehreren der vier Attribute top, left, right und bottom um einen bestimmten Wert verschieben. Wenn Sie diese Methode verwenden möchten, müssen Sie dies explizit angeben.

Beispiel

img#logo{
position:relative;
top:5px;
left:10px
}

dann wird das Element - gemessen von seiner Normalposition im HTML-Dokument - um 5px nach unten und 10px nach rechts verschoben.

Wenn Sie die Eigenschaft position:relative vergeben, ohne zusätzliche Angaben zu top, left, right oder bottom ändert sich nichts an der Darstellung , das Element bleibt an Ort und Stelle. Diesen Umstand werden werden wir uns später zunutze machen....

Absolute Positionierung

Ganz anders verhält es sich mit der absoluten Positionierung. Nachdem Sie diese Methode mit Hilfe des Attributs position:absolute für ein Element festgelegt haben, können Sie es mit einem oder mehreren der vier Attribute top, left right und bottem frei auf ihrer Seite platzieren.

img#logo{
position:absolute;
top:10px;
left:20px
}

Das so positionierte Element wird dabei aus dem normalen Objektfluss herausgenommen und auf eine höhere Ebene gesetzt. Es liegt also vor allen andernen Objekten und kann diese dadurch auch nicht mehr verdrängen.

Jedes Element, das Sie auf diese Weise positionieren, wird dabei auf eine eigene Ebene gesetzt, d.h mehrere absolut positionierte Elemente können sich auch überlagern oder überlappen.

Festlegen der Ebenenhierachie

Dabei gilt normalerweise das Element als das vorderste, das im HTML-Quelltext an letzter Stellesteht. Diese Ebenenhierachie können Sie jedoch mit dem Attribut z-index("Objektindex") und einem numerischen Wert ändern. je höher der Wert, desto weiter vorn liegt das Element:

img#logo{
position:relative;
top:5px;
left:10px
z-index:1;
}

h1#main-head{
position:relative;
top:5px;
left:10px
z-index:2;
}

Jede höhere Zahl setzt das entsprechende Element um eine Ebene nach vorne. Dabei müssen Sie die Ebenen nicht sequenziell numerieren, d.h auf z-index muss nicht zwingend z-index:2 folgen, sondern es kann gleich 5 oder 20 sein. Noch oben gibt es theoretisch keine Grenzen - Sie könnten einem Element also auch problemlos einen z-index von 9000 zuweisen. Experimente haben gezeigt, dass in den gängigen Browsern die Obergrenze aus technischen Gründen bei 2.147.483.647 liegt

NOTE: Der z-index kann auch bei position:relative verwendet werden, da sich auch auf diese Weise verschobene Objekte überlappen.

Festlegen des Bezugspunktes der Positionierung

Bei der absoluten Positionieeung mit top, left etc. gilt als Bezugspunkt das nächsthöhere Element, das nicht mit position:static positioniert ist. Falls kein von static abweichend positioniertes Vorfahrenelement existiert, bezieht sich die Positionierung auf das Browserfenster

NOTE: Da die relativen Positionierung keine Veränderung bewirkt, sofern keine Werte für top, left, right und bottom angeben werden, wird sie sehr häufig verwendet, um ein Element als Referenzobjekt für die absolute Positionierung eines seiner Nachfahrenelemente zu definieren.

Negative Werte

Mit position:absolute können Sie interessante Effekte erzeugen, wenn Sie für die positionsangeben negative Werte vergeben

Fixierte Positionierung

Eine vierte Positionierungsmöglichkeit ist position:fixed. Diese Positionierungart funktioniert ähnlich wie absolute, das Element bleibt jedoch an Ort und Stelle, selbst wenn der übrige Seiteninhalt gescrollt wird. Der Inhalt scrollt also hinter dem betreffenden Element hindruch.

Diese Methode ist sinnvoll, wenn Sie eine Navigation erzeugen möchten, die der Anwender imme im Blick hat, auch wenn der übrige Seiteninhalt gescrollt werden muss.
