Kapitel 9 Textumfluss.

Trivia

Wie wir wissen, werden in HTML-Elemente im Browser standardmäßig immer untereinander angeordnet. CSS bieten zahlreiche Möglichkeiten, dies zu ändern. Für die eigentsich hier der Textumfluss mit Hilfe des Attributs float

Dieses Attribute kann die Werte left und right annehmen. Wenn sie einem Element die Eigenschaft float:left zuweisen, dann wird dieses Element von allen nachfolgenden Elementen rechts umflossen , sofern der Platz dafür ausreicht. Bei float:right wird das Element links umflossen.

BSP: img{
float:left;
}

Allerdings wirkt sich das Attribut float auf alle nachfolgenden Elemente aus. Wenn sie diese nachfolgenden Elemente wieder über die gesamte Breite darstellen wollen, müssen Sie die float Anweisung für das erste dieser Elemente wieder deaktivieren Dies geschieht über das Attribut clear Das Attribut clear kann die Werte left(linken Umfluss deaktiviern), right(rechten Umfluss deaktivieren) und both (beide deaktivien) annehmen. Am einfachsten ist meist both:

BSP: img2{
clear:both;
}
